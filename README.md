# OSINT - Análise do Edifício do Consulado Norte-Americano em São Paulo

Este site realiza buscas automáticas de informações OSINT (Open Source Intelligence) sobre o edifício do consulado norte-americano localizado na Rua Henri Dunant 500, São Paulo, SP, 04709-110.

## Funcionalidades

- **Dados Geográficos**: Integração com a API GEOSAMPA para obter informações sobre edificações, riscos geológicos e infraestrutura urbana.
- **Informações de Segurança**: Links para sites oficiais do consulado e embaixada com informações sobre vistos e segurança.
- **Oportunidades de Modernização**: Análise baseada em dados OSINT para sugerir melhorias radicais em segurança predial.

## Como Usar

Abra o `index.html` em um navegador web. O site carregará automaticamente os dados.

## Tecnologias

- HTML/CSS/JavaScript
- API GEOSAMPA
- Nominatim (OpenStreetMap) para geocodificação

## Publicação no GitHub Pages

1. Faça push deste repositório para o GitHub.
2. Vá para as configurações do repositório.
3. Na seção "Pages", selecione a branch "main" e a pasta "/ (root)".
4. Salve e aguarde a publicação.

O site estará disponível em `https://<seu-usuario>.github.io/projeto-anti-consulado/`.
